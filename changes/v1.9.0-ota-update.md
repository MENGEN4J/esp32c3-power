# v1.9.0 OTA 固件更新

> 状态：已完成 | 分支：feature/v1.9.0 | 日期：2026-05-21

## Proposal

用户配网后，设备自动检查 Gitee 上的版本清单，发现新版本时用户在 Web 页面点击即可完成更新。更新失败时设备可自动回滚，不会变砖。

## Design

- 方案 C：文件级差量 OTA + 安全回滚
- 托管格式：.mpy 字节码（代码保护 + 下载更小 + 启动更快）
- 版本清单：Gitee Raw version.json（含 mpy_version 校验）
- 安全回滚：boot.py __import__ 校验 + .bak 自动回滚
- 详细设计文档：docs/ota-update-design.md
